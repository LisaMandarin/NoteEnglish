# Project Name: NoteEnglish 句句通
# Contributor: Min-ting (Lisa) Chuang

- NoteEnglish is a bilingual study assistant for English learners.
- Splits input text into sentences and preserves original order.
- Translates each sentence (Gemini) with normal or learner-friendly mode.
- Extracts key vocab (spaCy) and enriches it with definitions/examples/CEFR.
- React + Vite frontend with FastAPI backend; configurable API base and CORS.
- Ready-to-run local dev: npm/Node for UI, Poetry/uvicorn for API.

中文版：
- NoteEnglish 是為英語學習者打造的雙語學習助手。
- 將輸入文字拆成句子並保持原順序。
- 透過 Gemini 翻譯每句（一般/學習者友善模式）。
- 使用 spaCy 抽取重點單字，補充定義、例句、CEFR 等級。
- React + Vite 前端、FastAPI 後端；API 位址與 CORS 可調整。
- 本機開發直接使用 npm/Node 跑前端，Poetry/uvicorn 跑後端。
